# Stand up!
An homage to the great Chris Bridges.

Set this on `cron` to play when your team has stand-up its meeting!

![stand uppp](https://media.giphy.com/media/Och7EQ1EuemZO/giphy.gif)

### Requirements
- NodeJS >= 6.0.0
- Mac OS X/macOS >= 10.8
- Spotify for Mac
